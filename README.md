## CoinGame
CoinGame is a rng-based game, made in pygame.
![screenshoot](https://github.com/dawidekszopen/coinGame/blob/main/2024-10-22_18-20.png)

## How to open?
To open CoinGame you need to have installed
- pygame
- pygame-widgets

```
python main.py
```
